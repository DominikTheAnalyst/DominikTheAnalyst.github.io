Data exploration in SQL

In this data analysis and visualization project, I followed a structured approach to process and analyze COVID-19-related data from two tables, "covidDeaths" and "covidVaccination,". I conducted several data analyses to gain insights into the COVID-19 data, such as calculating death percentages, population percentages infected, and identifying countries with the highest infection rates. I joined the "covidDeaths" and "covidVaccination" tables using location and date as keys to analyze vaccination progress, specifically for European countries. A view population vaccination progress and percentage population vaccinated was created to store data for later visualization, allowing for easy access to vaccination progress data.
